FOUR STEPS TO PLAY
1. Purchase the ticket on ticketing screen
2.Change the ticket if u don’t like it and freeze the ticket
3. See the Numbers on Game screen
4. Let us know when you finish and event (when u have won any row or full )


Rules
a) Choose only one ticket per game and make sure gave your correct name while choosing user name
b) Once ticket freezes you can’t change the ticket
c)Always observe the game screen -don’t miss any number (selected number will turn green )
d) let us know as soon as you finish your row or full by sending in what’s app group


Note : To keep it easy take a paper and note down all the ticket numbers
